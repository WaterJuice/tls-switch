# tls-switch

A TLS switch utility.

## Features

- **Zero dependencies** — Python 3.12+ stdlib only

## Requirements

- Python 3.12+

## Quick Start

### Install

```bash
pip install tls-switch
```

Or run directly with uv:

```bash
uvx tls-switch
```

### Run

```bash
tls-switch hello
```

See the [Usage](usage.md) page for full details.
