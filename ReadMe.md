# tls-switch

A TLS switch utility.

## Features

- **Zero dependencies** — Python 3.12+ stdlib only

## Requirements

- Python 3.12+

## Installation

```bash
pip install tls-switch
```

Or run directly with uv:

```bash
uvx tls-switch
```

## Usage

```bash
tls-switch hello
```

## Development

```bash
# Set up development environment
make dev

# Run linting and type checking
make check

# Auto-format code
make format

# Build wheel and docs
make build
```
